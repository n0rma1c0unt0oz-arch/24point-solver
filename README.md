# 24点求解器（纯C实现）
基于穷举搜索与双栈求值算法实现的24点益智游戏，支持挑战模式、双人PK、暗黑记忆模式等6种功能，核心逻辑可类比自动驾驶轨迹搜索的空间枚举问题。

## 核心亮点
- **全量解搜索**：枚举4数字全排列（24种）+ 运算符组合（64种）+ 括号结构（5种），总搜索空间7680种可能，保证解完整性
- **高精度计算**：引入1e-6浮点容差机制，避免除法运算的精度误差
- **双栈表达式求值**：实现经典中缀转后缀求值算法，正确处理运算符优先级与括号嵌套
- **实时交互体验**：非阻塞限时输入、控制台闪烁特效、退格删除等交互逻辑
- **解法去重**：通过字符串匹配过滤重复解，避免相同表达式多次输出

## 功能列表
1. 挑战模式：10道题分3档难度，限时作答，自动计分
2. 自由练习：随机出题/自定义题目，支持随时查看正确答案
3. 双人PK模式：两人轮流对战，答对得分，答错/超时给对方加分
4. 暗黑模式：数字闪烁3秒后隐藏，凭记忆作答
5. 自动判题：校验表达式合法性、数字使用正确性，自动计算表达式结果

## 算法说明
### 1. 24点求解逻辑
采用「生成-验证」穷举策略：
- 数字全排列：递归交换实现4数字24种排列
- 运算符枚举：三重循环遍历3个运算符（+-*/共4^3=64种组合）
- 括号结构枚举：覆盖5种合法括号组合，保证无遗漏
- 浮点校验：通过`fabs(result-24) < 1e-6`判断结果正确性

### 2. 双栈表达式求值
- 操作数栈：存储中间计算结果
- 运算符栈：存储运算符与括号
- 优先级规则：`*/`优先级为2，`+-`优先级为1，遇右括号触发栈内运算直至匹配左括号

详细算法逻辑见仓库内`功能说明.txt`。
